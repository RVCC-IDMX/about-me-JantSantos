CSS ANIMATIONS

CSS animations allows a webpage developer to create animations without the use of JavaScript. CSS animations are very common and are always found in modern web design. The animations you can create are limitless because every property of the animation is modifiable from the element's appearance, position, color, shapes, rotations and much more. These designs elevates a webpage's design into a whole new level.

A user must create two(2) properties in order to create an animation for CSS which are animation-name which selects a specific animation and animation-duration which determines the duration of the animation. By using @keyframes rule in your CSS file, it is a way to gradually make a current style change from current onto another. A webpage developer can create smooth transitions/animations that changes the appearance of an element that you have created. IMAGINATION IS YOUR LIMIT!
